## 搭建vue开发环境
```
npm install -g @vue/cli 
vue --version
vue -V
vue -h
```

```
vue create vue-sell-cube
```

![Alt text](./vue-cli3.0安装过程.png)

#### 安装cube-ui
```
vue add cube-ui
```
![Alt text](./安装cube-ui.png)
- 在main.js中
```
import './cube-ui'
```
